Firebase Setup Guide — Membuat Project Baru Optional
Firebase yang terhubung sekarang sudah aktif dan data realtime berjalan. Panduan ini untuk referensi jika perlu membuat project Firebase baru.
01
Buat Project Firebase
Buka console.firebase.google.com, login dengan akun Google, lalu klik Add project. Beri nama project, nonaktifkan Google Analytics jika tidak diperlukan, lalu klik Create project.
02
Aktifkan Realtime Database
Di sidebar Firebase Console, pilih Build → Realtime Database, klik Create Database. Pilih region asia-southeast1 (Singapore) untuk latensi rendah dari Indonesia, lalu pilih Start in test mode agar bisa read/write dulu.
Test mode hanya berlaku 30 hari. Setelah itu, buat rules yang lebih ketat untuk produksi.
03
Daftarkan Web App
Di halaman utama project, klik ikon Web (</>) untuk mendaftarkan app. Beri nama app, lalu klik Register app. Firebase akan menampilkan firebaseConfig seperti ini:
apiKey: "AIzaSy...", authDomain: "nama-project.firebaseapp.com", databaseURL: "https://nama-project-default-rtdb.firebaseio.com", projectId: "nama-project", storageBucket: "nama-project.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abcdef"
04
Update Config di script.js & config.h
Ganti nilai firebaseConfig di bagian atas script.js dengan config baru. Di Arduino, buka config.h dan ganti nilai FIREBASE_DB_URL:
#define FIREBASE_DB_URL "https://nama-project-default-rtdb.firebaseio.com" #define DEVICE_ID "esp32-ph-01"
05
Set Database Rules
Pergi ke Realtime Database → Rules. Untuk development, gunakan:
{ "rules": { ".read": true, ".write": true } }
Jangan gunakan rules terbuka di production. Tambahkan autentikasi atau batasi IP setelah testing selesai.
06
Verifikasi Koneksi
Upload kode ke ESP32, buka Serial Monitor di Arduino IDE (baud 115200). Tekan tombol scan, pastikan muncul log [Firebase] SUCCESS!. Di dashboard ini, indikator pojok kanan atas harus berubah menjadi ONLINE dan data muncul secara realtime.
